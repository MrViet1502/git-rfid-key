## Used devices:
- Arduino pro mini 3v3
- MFRC522 and Cards
- Button config
- Button Power
- Buzzer
- USB UART
  
## Wiring connection diagram

- RED and Black big -> 48V
- White of Button (Pin 6)
- Graay of RESET (Pin reset)
- Blue and white big of Relay 
  ### Load code by USB UART
  - Yellow of TX
  - Green of RX
  - Black small of GND
